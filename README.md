# PlanIA — Página de Vendas

Landing page low-ticket da oferta **PlanIA**: ensina a criar planilhas e dashboards profissionais usando Inteligência Artificial (ChatGPT, Claude, Gemini).

## Conteúdo do repositório

- `pagina-de-vendas.html` — página de vendas completa e autossuficiente (HTML + CSS embutido + JS, sem build).
- `imagens/` — logo (`logo-plania.png`) e imagem do hero.
- `planilhas exemplos/` — imagens de painéis exibidas no carrossel de exemplos.
- `oferta-planilhas-ia.md` — documento estratégico da oferta (avatar, copy, preços, e-mails).

## Como visualizar

Abra o arquivo `pagina-de-vendas.html` direto no navegador.

Ou rode um servidor local na pasta do projeto:

    python3 -m http.server 5500

E acesse http://127.0.0.1:5500/pagina-de-vendas.html

## Personalizar

Procure por `[PREENCHER]` e `[LINK CHECKOUT]` no HTML para inserir o link de checkout, o vídeo e os depoimentos reais.
